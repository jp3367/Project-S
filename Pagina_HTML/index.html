¿Qué es PROYECT S?

PROYECT S es una página web dedicada sobre nuestro sistema solar. Podrás encontrar información sobre cada planeta, como su tamaño, temperatura, y se añadirán algunos detalles curiosos de cada planeta del sistema.

Este proyecto se irá actualizando mediante pase el tiempo, ya sea con actualizaciones de diseño de la página, cambios de logo, nuevos detalles de cada planeta e incluso se añadirán nuevas secciones a lo largo del tiempo.

Esta versión es una beta, al ser el primer modelo de la página es probable encontrar errores, bugs, etc.
